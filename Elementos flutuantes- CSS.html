Position: static

Um elemento com posição static não está posicionado de maneira especial, isto é, estará sempre posicionado de acordo com o fluxo normal da página.

Position: relative

O elemento é posicionado de acorod com o fluxo normal do documento, e, em seguida, deslocado em relação à própria base nos valores de top, right, bottom, e left.

este elemento foi posicionado com relative.

Position: Absolute

Um elemento com posição absoluta está posicionado em relação ao antepassado posicionado mais próximo.

Position: fixed

Um elemento com posição fixed está posicionado em relação à viewport, o que significa que ele fica sempre no mesmo lugar, mesmo que a página seja rolada.
